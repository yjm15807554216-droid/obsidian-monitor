# 监控指标详细说明

> 返回首页：[[00_告警追踪系统_首页]]

## 1. 控筹率 (OI / Market Cap)

### 定义

```
控筹率 = 合约持仓总价值 (USD) / 流通市值 (USD)
       = (Open Interest 张数 × 当前价格) / Circulating Market Cap
```

### 含义

衡量"合约市场上的杠杆持仓"占"流通币总价值"的比例。

**通俗理解：** 流通在外的每 100 个币里，有多少个被拿去做合约杠杆交易？

### 经验阈值

| 数值 | 状态 | 含义 |
|------|------|------|
| < 5% | 🟢 健康 | BTC、ETH 等主流币常态 |
| 5-10% | 🟢 正常 | 热门主流币日常水平 |
| 10-15% | 🟡 偏高 | 开始出现杠杆味道 |
| 15-20% | 🟠 异常 | 需要警惕，杠杆资金偏多 |
| 20-30% | 🔴 高风险 | 庄家可能在布局或准备收割 |
| > 30% | 💀 极度危险 | 随时可能出现暴动 |

### 局限性

- 不同币的"正常水平"差异很大
- 大市值币天然控筹率低（无法控盘）
- 小市值新币天然控筹率高
- **绝对值意义有限，相对值（分位数、变化率）更有信号价值**

## 2. 历史分位数 (Historical Percentile)

### 定义

```
分位数 = 当前控筹率在过去 N 天数据中的相对位置
当前窗口 = 14 天
```

### 含义

修正"绝对值"的局限。同样是 17%，对于平均值 5% 的币是异常，对于平均值 18% 的币是常态。

**举例：**
- BTC 控筹率 12%（历史分位 99%）→ 极度异常
- PEPE 控筹率 25%（历史分位 50%）→ 它的常态

### 优点

- 自适应每个币的特征
- 排除"长期高控筹但稳定"的噪音
- 能识别"突破常态"的真信号

### 局限性

- 需要至少 20 个数据点才能计算
- 数据断档会影响准确性
- 新上线的币没有足够历史数据

## 3. 变化率 (Rate of Change)

### 定义

```
1h 变化率 = 当前控筹率 - 1 小时前的控筹率
4h 变化率 = 当前控筹率 - 4 小时前的控筹率
```

### 含义

衡量控筹率的"变化速度"，捕捉**资金涌入的瞬间**。

**为什么重要：** 同样是 25% 的控筹率，"突然涨上来的"比"长期稳定的"危险得多。前者意味着资金正在当下涌入，后者只是常态。

### 经验阈值

| 1h 变化 | 含义 |
|---------|------|
| < +2pp | 正常波动 |
| +2 ~ +5pp | 资金加仓 |
| +5 ~ +10pp | 急速涌入（告警） |
| > +10pp | 暴力涌入（极度异常） |

> "pp" = percentage points（百分点）

## 4. 资金费率 (Funding Rate) — *待加入*

### 定义

币安永续合约的多空平衡机制。每 8 小时结算一次（部分合约 4 小时）。

```
正费率：多头付钱给空头（多头偏多）
负费率：空头付钱给多头（空头偏多）
```

### 含义

最直接的多空对决指标。**资金费率越极端，市场情绪越偏激，未来波动越大。**

### 经验阈值

| 资金费率 (8h) | 状态 |
|-------------|------|
| ±0.01% | 正常 |
| ±0.05% | 偏极端 |
| ±0.1% | 异常 |
| ±0.5% | 严重异常 |
| ±1% | 极度危险（不可持续） |

### 与控筹率的关系

**真正有价值的信号是两者的组合：**

| 控筹率 | 资金费率 | 解读 |
|--------|---------|------|
| 高 | 高正 | 多头堆积，警惕回调 |
| 高 | 高负 | 空头堆积，警惕轧空 |
| 高 | 中性 | 多空平衡，方向不明 |
| 低 | 极端 | 杠杆少，影响有限 |

---

## 5. 复合信号 — *未来扩展*

最强的告警一定是**多个独立指标同时触发**。例如：

```
控筹率 14d 分位 ≥ 95%
+ 资金费率 |FR| ≥ 0.5%
+ 24h 成交量 ≥ 历史均值 × 3
+ 价格 24h 变化 ≥ 30%
= 🆘 极度危险信号（多空大对决一触即发）
```

这种复合信号的胜率远高于单一指标，但触发频率也低得多。**质量 vs 数量的权衡。**
